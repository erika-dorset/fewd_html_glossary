Lists

There are 2 most common types of lists:

<ol> - Ordered Lists
<ul> - Unordered Lists

The list items will be marked with bullets (small black circles) by default for an unordered list and numbered for an ordered list.

An Ordered HTML List

  1. Coffee
  2. Tea
  3. Milk

An Unordered HTML List

  • Coffee
  • Tea
  • Milk

Code:
<h2>An Ordered HTML List </h2>
<ol>
<li> Coffee </li> 
<li> Tea </li> 
<li> Milk </li>
</ol> 

<h2>An Uordered HTML List </h2>
<ul>
<li> Coffee </li> 
<li> Tea </li> 
<li> Milk </li>
</ul>

Definition List

The <dl> tag defines a description list.

The <dl> tag is used in conjunction with <dt> (defines terms/names) and <dd> (describes each term/name).


Here is an example of a definition list:

Coffee
Black hot drink
Milk
White cold drink


Code:
<p>Here is an example of a definition list:</p>
<dl>
<dt> Coffee </dt>
<dd> Black hot drink </dd>
<dt> Milk </dt>
<dd> White cold drink </dd>
</dl>
