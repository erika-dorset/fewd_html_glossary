Semantic tags for Text

The <p> element is used for paragraphs. You can make text Bold by using the <b> element and italic by using the <i> element.


For mathematical concepts concepts or suffixes of dates you can use the <sup> element.

For example: 

<p>Today is 23<sup>rd</sup> of September 2020. </p>
Result: Today is 23rd of September 2020.


You can add a line break by using the <br/> element.
And a create a break between themes (line) by using the <hr/> element.

There is also an element which is used for chemical formulas.

For example: 
<p>You should drink your H<sub>2</sub>O because it's good for you! </p>
Result: You should drink your H2O because it's good for you!

Semantic Markup

Semantic Markup doesn't affect the structure of web pages however it adds extra information that different browsers will interpret.

<strong> Shows that the content has strong importance (Makes the text bold).

For example: 

<p>This is <strong>important</strong> text. </p>
Result: This is important text.


<em> Emphasis that subtly changes the meaning of a sentence. Browsers show in italic by default.

For example: 
<p>Let's make this text <em>italic</em>... </p>
Result: Let's make this text italic ...

You can use <blockquote> to add long quotations. And <cite> for references. The browser will render it in italics. For the blockquote opening tag make sure to use the cite="URL" attribute to specify where the source is from.

For example: 
<blockquote cite="https://blog.hatchcoding.com/6-great-quotes-about-the-power-of-coding">

I think it's fair to say that personal computers have become the most empowering tool we've ever created. 
They're tools of communication, they're tools of creativity, 
and they can be shaped by their user.

<cite>- Bill Gates</cite>
</blockquote>
Result:
I think it's fair to say that personal computers have become the most empowering tool we've ever created. They're tools of communication, they're tools of creativity, and they can be shaped by their user.
- Bill Gates


<q> Is used for short quotes.

For example: 
<q>Everybody should learn to program a computer, because it teaches you how to think.</q> - Steve Jobs
Result: Everybody should learn to program a computer, because it teaches you how to think. - Steve Jobs


<abbr> Is used for abbreviations or acronyms.You need to use the title="value" attribute to specify what the abbreviation stands for.

For example: 
<p>Hover over <abbr title="Hypertext Markup Language">HTML</abbr> to see what it stands for! </p>
Result: Hover over HTML to see what it stands for!


<address> Element defines the contact information of the author. By using the href="mailto:email" attribute will open a default mail program, create new message with the TO field already filled out.

For example: 
<address>Written by <a href="mailto:email@example.com">Erika Volodko</a>.<br/>Visit us at:<br/>Example.com<br/>Box 564, Disneyland<br/>USA
Result:
Written by Erika Volodko.
Visit us at:
Example.com
Box 564, Disneyland
USA


<ins> Shows inserted content while <del> shows deleted content.

For example: 
<p>This text has been <ins>inserted</ins> while this text has been <del>deleted</del>.</p>
Result: This text has been inserted while this text has been deleted.


<s> Indicates that something isnt relevant but should still be there.

For example: 
<p>This text isnt <s> relevant </s> but should still be there.</p>
Result: This text isnt relevant but should still be there.
